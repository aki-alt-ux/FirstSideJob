最初のWebサイト

HTMLとは

HTMLは、ハイパーテキストマークアップの略で、
Web上の文書を記述するためのマークアップ言語です。

Webサイトで使用する言語

HTML
Webページにおける文書(情報のまとめ役)を担当します。
CSS
Webページにおけるデザイン(見た目を整える役)を担当します。
JavaScript
HTMLやCSSへの動的な操作をするために使います。
